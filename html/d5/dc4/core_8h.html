_key_name,

值:
({\
_type * _pos = NULL;\
_type * _ret = NULL;\
rcu_read_lock();\
list_for_each_entry_rcu(_pos, &_head, _list_name) {\
if (_compare(_pos->_key_name, _key) == 0) {\
_ret = _pos;\
break;\
}\
}\
rcu_read_unlock();\
_ret;\
})

Find an entry in a doubly linked list according to one key in the struct.

To support polymorphism. This can be applied to different structs with struct list_head and searching according to different keys. Since this uses RCU and is the read-side, do not use this macro inside any locks.

参数
_typetype of the struct
_key_namename of the key within the struct
_keyvalue of the key of the entry to find
_headstruct list_head. The head of the list
_list_namename of struct list_head within the struct
_compareint (*_compare)(KeyType key1, KeyType key2). The pointer to the function to compare keys. Return 0 if equal.
返回
pointer to the first entry found. If not found, return NULL.

int _result = 0;\
if (_min <= _value && _value <= _max) {\
_result = 1;\
} else if (_min > _max) {\
_result = 1;\
} else {\
_result = 0;\
}\
if (_is_exclude) {\
_result = (_result + 1) % 2;\
}\
_result;\
})

Match value in an interval or match value not in an interval.

To support polymorphism like template. When min_value > max_value, it means all the values.

返回
1: MATCH, 0: NOT_MATCH
